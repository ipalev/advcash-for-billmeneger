Для установки модуля необходимо:
1. Разложить файлы из папок в соответствующие(с таким же названием) папки в корневом каталоге билменеджера.
2. Обновить панель билменеджера, чтоб она перезагрузилась (можно установить или удалить какой-нибудь модуль оплаты), тогда в списке модулей появится новый модуль AdvCash.
3. Теперь можно создавать метод оплаты из этого модуля. 
	1) Укажите мал кошелька, 
	2) Название магазина, которое вы указывали при регистрации SCI 
	3) url возврата должен состоять из домена билменеджера плюс ссылка на файл обработки результата, у вас должно быть так: https://my.justvps.com/mancgi/AdvCashresult.php это ссылка на статус, успешную и ошибочную страницы оплаты, (обработка идет в одном файле не стал городить три поля для одинаковых ссылок)
	4) Пароль SCI, если используете, его вы задаете сами когда создаете SCI в кабинете AdvCash, если не используете, оставьте поле пустым, оно не обязательное, хотя почему-то с красной звездочкой отображено.
	5) 6) Название и пароль АПИ тоже задаете в кабинете AdvCash, нужно для рефандов.
4. На втором шаге создания модуля заполняете стандартные поля, первой и второй строкой идет название в локациях, это название будет отображено в списке способов оплаты когда клиент переходит к оплате услуг. Остальные поля не комментирую.
5. После нажатия кнопки завершить новый метод оплаты будет добавлен, выбираете его и жмете изменить и можете добавить иконку на ваш способ оплаты их там две для разных случаев, если нужно поставьте галочку напротив "Разрешить возврат платежей", для выполнения рефандов. Обращаю внимание при выполнении вывода средств клиентам нужно указывать платеж, а не оставлять "все неизрасходованные", это связано с тем что с платежа подтягивается необходимое поле почты клиента, если ну указать платеж выдаст ошибку.
Все, будут вопросы обращайтесь.
